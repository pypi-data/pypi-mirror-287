# AxArith
Verilog Library of Approximate Arithmetic Operators
## Circuits
### Approximate Adders
* CopyA and CopyB
* Trunc0 and Trunc1
* LOA (Lower-part Or Adder)
* ETA-I (Error-Tolerant Adder 1)
### Approximate Multipliers
* LoBA [1]
* DRUM [2]
